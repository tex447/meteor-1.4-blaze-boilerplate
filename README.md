# Boilerplate: Meteor 1.4 + Blaze

Build with Flow Router, UserAccount:core, Bootstrap, Collection2, Autoform, Users Roles, Cluster Micro Services

To install the App Structure just run:
npm install
meteor

**Next Features:**
